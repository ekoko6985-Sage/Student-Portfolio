## 📋 Project Overview

A modern, responsive, professional student portfolio website showcasing Emmanuel Koko's cybersecurity projects, skills, and academic achievements. Built with pure HTML5, CSS3, and Vanilla JavaScript—no frameworks or libraries.

This project was created as a comprehensive submission for the **COS 106 Practical Term Project** at Miva Open University, demonstrating professional web development standards and best practices.

## 🎯 Key Features

### 🌐 Responsive Design
- Fully responsive layout optimized for desktop, laptop, tablet, and mobile devices
- Mobile-first approach with CSS Grid and Flexbox
- Hamburger menu navigation for mobile devices
- Touch-friendly interface elements

### 🎨 Modern UI/UX
- **Glassmorphism** effects for premium look and feel
- Beautiful gradient backgrounds and smooth shadows
- Smooth animations and transitions throughout
- Professional color scheme: Blue (#0D6EFD) and Orange (#FF8C00)
- Custom CSS variables for easy theme management

### 📄 Multi-Page Website
- **Home Page**: Hero section, statistics, skills preview, projects showcase
- **About Me Page**: Educational timeline, technical skills, career aspirations, personal interests
- **Projects Page**: Project showcase with filtering, modal popups, technology tags
- **Academic Planner**: Fully functional task manager with local storage
- **Contact Page**: Contact form with validation, FAQ accordion

### ⚡ Interactive Features
- **Academic Planner**: Add, edit, delete, search, and filter tasks
- **Contact Form**: Real-time validation with error messages
- **Project Filtering**: Filter projects by category
- **FAQ Accordion**: Interactive frequently asked questions
- **Animated Counters**: Statistics with smooth animations
- **Scroll Animations**: Elements fade in as you scroll
- **Back to Top Button**: Smooth scroll to top
- **Scroll Progress Bar**: Visual indication of page scroll position

### 💾 Local Storage
- Tasks persist across browser sessions
- Form submissions stored locally for demonstration
- No backend required

### ♿ Accessibility
- Semantic HTML structure with proper heading hierarchy
- ARIA labels and attributes for screen readers
- Keyboard navigation support
- High contrast text for readability
- Alt text for all images
- Skip to main content link

### 🔍 SEO Optimized
- Meta tags for description and keywords
- Proper semantic HTML
- Structured content
- Mobile-friendly design

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No installation or dependencies required
- Works offline

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ekoko6985-Sage/Student-Portfolio.git
   cd Student-Portfolio
   ```

2. **Open in browser**
   - Double-click `index.html` to open locally
   - Or use a live server extension in VS Code

3. **Deploy to GitHub Pages**
   - Push to GitHub repository
   - Enable GitHub Pages in repository settings
   - Set source branch to `main`
